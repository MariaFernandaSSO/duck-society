# 🦆 The Duck Society

*Leia em [English](README.md) • [Português](README.pt.md)*

Blog pessoal da Maria Fernanda. Feito com HTML, CSS e JavaScript puro. Hospedado no GitHub Pages.

## Sobre

Um lugar pra escrever sobre desenvolvimento backend, sistemas de pagamento, microsserviços e o que mais eu estiver aprendendo ou quebrando no momento. Também patos.

## Stack

- HTML + CSS + JavaScript (sem frameworks, sem build step)
- GitHub Pages

## Estrutura

```
duck-society/
├── index.html
├── 404.html
├── pages/
│   ├── blog.html
│   └── about.html
└── assets/
    ├── css/
    ├── js/
    │   └── main.js
    ├── fonts/
    └── images/
```

## Funcionalidades

- **i18n** — alternar PT/EN com persistência em localStorage
- **Tema** — claro/escuro
- **Fonte** — controle de tamanho e família
- **Blog** — filtros por categoria, data e ordenação

## Desenvolvimento local

Sem build tools. Clone e abra o `index.html` no navegador.

```bash
git clone https://github.com/MariaFernandaSSO/duck-society.git
cd duck-society
# abra o index.html
```

## Deploy

Pushes na branch `main` são publicados automaticamente via GitHub Pages.
